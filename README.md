## ✨ Key Features & Architecture

### 1. Hero Section — Data → Intelligence
- Custom particle cluster system built with **Three.js / React Three Fiber**.
- Real-time cursor reactivity and dynamic rotation.

### 2. Interactive Insight Flow
- 3-stage data transformation walkthrough (*Ingest Data* $\rightarrow$ *Analyze with AI* $\rightarrow$ *Generate Insight*).
- Scroll-based viewport animations powered by **Framer Motion**.

### 3. Intelligence Dashboard Preview
- Mock product UI with high visual hierarchy, sidebar navigation, real-time metrics, and animated tab switching.

### 4. Signature Interaction (WOW Moment)
- Dynamic 3D Neural Mesh featuring fluid distortion physics on hover and wireframe state toggles on user click.
